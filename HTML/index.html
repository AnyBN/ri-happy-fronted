Atributos HTML: id: Identificador único de um elemento. style: Formata o HTML diretamente no elemento, utilizando comandos CSS. class: Padroniza o estilo de vários elementos com um mesmo identificador de classe. Elementos HTML: label: Define um rótulo para elementos de formulário, como caixas de texto, caixas de seleção, botões de rádio, etc. O atributo for vincula o rótulo ao id do campo de formulário correspondente. input: Abre uma caixa de texto. O atributo type é específico para definir o tipo do campo, como: text: Para texto. number: Para valores numéricos. color: Psara selecionar cores. img src="": Insere uma imagem. O link da imagem é colocado dentro das aspas "", e você também pode definir a largura da imagem. h1, h2, h3, h4, h5, h6: Definem títulos, variando de h1 (maior) a h6 (menor). p: Define parágrafos. blockquote: Insere uma citação com espaçamento. sup: Texto elevado (sobrescrito). sub: Texto inferior (subscrito). mark: Destaca o texto, como se estivesse grifado. i: Aplica estilo em itálico ao texto. u: Aplica sublinhado ao texto. Tipos de listas: Lista ordenada (ol): Usa números ou letras como marcadores. Cada item deve estar dentro de uma tag
  • (list item). Lista não ordenada (ul): Usa marcadores não numéricos (como pontos ou discos) e também utiliza a tag
  • . Links (ancoras): a href="": Cria um link. Atributo target: Define onde o link será aberto. target="_self": Abre o link na mesma aba. target="_blank": Abre o link em uma nova aba. title="": Define um título descritivo para o link. Estruturação semântica: Semântica: Forma de estruturar o HTML para facilitar a compreensão visual e a acessibilidade do site.
