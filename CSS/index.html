Bem-Vindo!

No CSS nós possuimos os seletores por ID

  • Seletor por tipo: Busca elementos por uma tag HTML.
  • Seletor por ID(ª): Busca elementos através do atributo "id".
  • Seletor por Chave(.): Busca elementos atrvés do atributo "class".
  • Seletor Universal(*): Seleciona todos os eementos HTML.
  • Seletor de atributo ([atrib]): Seleciona elementos que possuem um atributo especifíco em nosso documento HTML e conseguimos também buscar atributos com um valor especifico.
